In this problem of leetcode the user gives an element which will have to be removed from the given input of multiple numbers.
Example:-
a=[1,2,2,3,4,2,5]
val=2
new array : [1,3,4,5]
So,in the given example as we can see that "val" (the element we have to remove) from "a" (the given numbers).
Here we remove the "val" element as many time as it appears in the "a" (given numbers).
